# Автоматизированная система обработки корпоративной почты

Система для классификации входящих обращений отдела IT-поддержки. Приложение читает входящие обращения из файловой системы (`inbox/`), классифицирует их по смысловым категориям и перемещает файлы в соответствующие папки, после чего формирует отчёт о проделанной работе.

## Категории классификации
* `urgent` — Важные/Срочные инциденты (содержащие ключевые слова типа "падает", "срочно", "инцидент").
* `alerts` — Автоматические уведомления от систем мониторинга (Grafana, Zabbix).
* `spam` — Спам и реклама.
* `hr_documents` — Кадровые вопросы, отпуска, больничные и согласование документов.
* `newsletters` — Корпоративные дайджесты и новостные рассылки.
* `support_requests` — Обычные запросы пользователей в поддержку (работает как fallback).
* `errors` — Ошибки (неверный формат файлов, пустые письма или поврежденные данные без базовых заголовков).

## Требования
- Python 3.12+
- `dateparser`

## Установка

При использовании пакетного менеджера `uv`:
```bash
uv sync
```

Либо с помощью `pip`:
```bash
pip install dateparser pytest
```

## Запуск

### Запуск через Bash-скрипт (рекомендуемый способ)
Bash скрипт выполняет проверки наличия директории, запускает python-код и выводит результаты статистики.
```bash
bash run.sh
```

### Запуск напрямую
```bash
python classifier.py
```

## Тестирование
Код покрыт юнит-тестами с использованием фреймворка `pytest`.

Для запуска тестов используйте:
```bash
uv run pytest tests/
# или
python -m pytest tests/
```

## Архитектура
* `structures.py` — класс `Letter`, который отвечает за парсинг текста письма, обработку заголовков, извлечение метаинформации (дата, отправитель, тема) и обработку исключений.
* `classifier.py` — основная логика работы. Скрипт перебирает файлы, проверяет их целостность, вызывает классификацию, перемещает файлы в итоговые директории и собирает статистику.
* `run.sh` — Bash скрипт, автоматизирующий вызов классификатора.
* `tests/` — модульные тесты для проверки корректности присвоения категорий и обработки граничных случаев (например, пустых писем).
